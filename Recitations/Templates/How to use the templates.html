1. Download the template you want to use
  2. Unzip the template
  3. Either create a copy of the template with the desired project name or rename the template folder itself
  4. Open with VSCode

I would recommend using the optimized template, as it can significantly reduce the build time. If you encounter any issues with specific features not working for the problem statement you're addressing, be sure to check the .mbedignore file. The optimized template removes many unused libraries to streamline the recitations.
